# fleishman
A Node-powered command-line tool for tasks related to front-end development at FleishmanHillard.

## install
Run ```npm install fleishman -g``` to install.

## usage
From Terminal (or your bash terminal of choice), run ```fleishman```.

To not have to go through the prompts, you can create a fleishman.yaml file in your Sites directory (or wherever you put all of your web projects). There is an example yaml file in this repository. **Make sure that your yaml file contains a value for all the options.**

Note: currently only supports Mac workflows.
